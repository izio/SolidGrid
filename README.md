SolidGrid
=========
# A responsive grid system for the rapid development of mobile friendly websites.

## Installation

### 1. Download the files
[Download the zip folder](http://www.izio.co.uk/#downloads), extract the files and copy the stylesheets in to your project folder.

### 2. Link the stylesheets
If you want a fluid grid add the following:
```html
<link rel="stylesheet" href="fluidgrid.css">
```
or alternatively for a fixed grid:
```html
<link rel="stylesheet" href="fixedgrid.css">
```
and optionally if you want to use the base stylesheet
```html
<link rel="stylesheet" href="base.css">
```
### 3. Viewport scale
Add the following meta tag to your HTML’s `<head>` section:
```html
<meta name="viewport" content="width=device-width,initial-scale=1">
```
Without this meta tag mobile devices may not resize the content to the actual view port width
