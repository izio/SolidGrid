Solid Grid

Solid Grid is a mobile first responsive grid framework that makes it easy to develop mobile friendly websites.

Standard Grid

Creating a new grid is simple, add a new container to house your columns and add the class .row, you can now add up to twelve .columns or .column classes to house your content.

Typography

Typography is based on the golden ratio (http://www.pearsonified.com/typography/) with a font size of 14px and line-height of 26px with the padding and margins of all standard elements set at multiples of 13px to ensure that your content has a natural vertical rhythm. The base stylesheet is entirely optional but we feel that it gives you an excellent starting point for some great typography.

There are also four classes that you can add to an image to change the way that they are displayed and can be combined in any combination.

  • The .scale class ensures that the image will not exceed the bounds of its container.
  • The .bordered class adds a white border and subtle drop shadow to the image.
  • The .rounded class adds rounded corners to the image.
  • The .circular class makes the image circular by adding a 50% radius border.

Tables

By default there are no formatting styles associated with the table tag this is to ensure that no styling is improperly associated with tables created by third party plugins such as calendars. However, by adding the class .table to a table its captions, headers and columns will all be styled as well as having zebra striping on alternate rows.

Forms

Elements within a form should be laid out using an ordered or unordered list and by default all form inputs except checkboxes and radio buttons are designed to utilise the full width of their parent element with labels being positioned above their related input.

To display an inline form with the label either to the left or right of the input apply add the class .left or .right to the label and set a width on the relevant input.

Email 
<form id="email-form">
    <fieldset>
        <legend>Email</legend>
        <ol>
            <li>
                <label for="email-form-title">Title</label>
                <select id="email-form-title" name="select">
                    <option value="1">Mr</option>
                    <option value="2">Miss</option>
                    <option value="2">Mrs</option>
                </select>
            </li>
            <li><label for="email-form-name">Name</label><input id="email-form-name" name="name" type="text" maxlength="255" /></li>
            <li><label for="email-form-email">Email</label><input id="email-form-email" name="email" type="text" maxlength="255" /></li>
            <li><label for="email-form-subject">Subject</label><input id="email-form-subject" name="subject" type="text" maxlength="255" /></li>
            <li><label for="email-form-message">Message</label><textarea id="email-form-message" name="message" row="5" cols="50"></textarea></li>
            <li class="text-right">
                <button name="cancel" type="button">Cancel</button>
                <input name="submit" type="submit" value="Submit" class="primary" />
                <input name="disabled" type="submit" value="Disabled" disabled="disabled" class="primary" />
            </li>
