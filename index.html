Tables

By default there are no formatting styles associated with the table tag this is to ensure that no styling is improperly associated with tables created by third party plugins such as calendars. However, by adding the class .table to a table its captions, headers and columns will all be styled as well as having zebra striping on alternate rows.

Forms

By default form inputs are designed to utilise the full width of their parent element with labels being positioned above their related input. To display an inline form with the label either to the left or right of the input apply add the class .left or .right to the label and set a width on the relevant input.

Content Boxes

There are four multipurpose boxes that can be added to your code to highlight content, simply create a div element with the class .box and optionally one of the following classes .information, .confirmation, .warning. Content boxes can also be further enhanced by adding the class .rounded which will add rounded corners to the element, and you can also add a title to the content box by adding an element with the class .title which will add a colour coordinated title bar.

Paging

Any ordered or unordered list can be converted in to a pager by simply adding the class .paging to the parent list, the current page is highlighted by adding the class .current to the list element which also prevents it from being clicked.

Images

There are three classes that you can add to an image to change the way that they are displayed and can be combined in any combination.

Scaled

Adding the class .scale ensures that the image will scale with its parent container.

<img src="..." class="scale"/>

Bordered

Adding the class .bordered adds a thin border and box shadow.

<img src="..." class="scale bordered"/>

Rounded

Adding the class .rounded adds rounded corners to the image.

<img src="..." class="scale rounded"/>
